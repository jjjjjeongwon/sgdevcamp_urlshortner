## 📝 주요 작업 및 기능
- URL 입력폼 제공
- 단축 후 결과 출력
- 동일한 URL을 입력할 경우 **항상 동일한 shortening 결과 값** 출력
- shortening 의 결과 값은 **8문자 이내**로 생성
- 브라우저에서 shortening URL을 입력하면 **원래 URL로 리다이렉트**
- 도메인은 **localhost**로 처리
